Key Commands

'Z' = Create New Cell With Random Color
'X' = Remove Cells
'A' = Match predrawn cell color.
'S' = Measurement.
'Q' = Draw Line
'W' = Line Eraser.
'F' = Move Cells with same color.
'D' = Copy and paste new cell with same area.
'G' = Move All Drawings.



About

Hi,
The tool is developed as minimal and distraction free as possible to help designers to focus on design instead of spending significant amount of time in learning it. The tool is easy to use and could be easily used by any person regardless background and intention.

Major uses of this tool could be anything from planning, elevation design, interior, pixelart, detail section, facade design, infographics, layout design, and on and on.

We are heading towards development of a complete automated solution with submission drawings package in one go.

Known Bug Notes

Following are few precautions you may have to take while using automation and shuffle commands for best user experience.
1. Once space is drawn. If you draw another space over it, the existing one gets removed. There will always be a single cell per grid regardless modifications.
2. Our intention here is to automate 80% initial job and assist designers to develop a new paradigm.
3. This tool is not replacement of any other tool but developed to develop a basic understanding of the project.
4. Moving and copying the spaces could be done for minimum 2 grid spaces.
